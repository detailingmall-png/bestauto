---
name: Always check if changes affect other pages
description: When editing blocks or features, always check if they exist on other pages and propose changes everywhere
type: feedback
---

When editing any block, schema, widget, or feature on bestauto.ge, ALWAYS check if the same element exists on other pages and propose/apply changes across ALL affected pages.

**Why:** User pointed out that Review schema was only on the homepage while the reviews widget was on all service pages. This caused Google Search Console errors on 36 pages. Changes should be consistent across the entire site.

**How to apply:**
1. Before editing a block: grep for similar blocks across all pages
2. When fixing a schema/SEO issue: check all page types (homepage, services, blog)
3. When adding a new feature to one page: ask if it should be on others too
4. Always propose a site-wide fix, not a single-page fix
